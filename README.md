Overview

LangNest is a web application designed to help users learn and practice different languages through an interactive and user-friendly platform. It provides structured learning modules, practice exercises, and progress tracking to enhance language learning efficiency.

Features

*User Features
-User registration and login (JWT authentication)
-Explore different languages
-Access learning modules and lessons
-Practice exercises and quizzes
-Track learning progress

*Admin Features
-Manage languages and lessons
-Add or update course content
-Monitor user activity

Tech Stack
Frontend
React.js
Axios
React Router
Tailwind CSS / Bootstrap

Backend
